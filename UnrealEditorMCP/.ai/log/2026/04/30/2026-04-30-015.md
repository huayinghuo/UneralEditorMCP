# 操作日志：2026-04-30-015

## 操作类型
阶段 11A：受控 Blueprint 图编辑（优先恢复）—— 6 Handler + 2 辅助文件

## 背景
参考项目 `UeAgentInterface` 的 Blueprint 实现方式：受控命令面（CmdBlueprintCreate / AddEventNode / AddCallFunctionNode / ConnectPins / Compile）。当前项目先做 11A 受控模板式 Graph MVP，再做 11B 声明式图重建。

## 6 个 MCP Action

### 1. blueprint_create_actor_class（Write）
- 创建 Actor Blueprint 类（父类必须 Actor 派生）
- 复用原有 `create_blueprint` 逻辑：AssetRegistry 冲突检测 → CreatePackage → FKismetEditorUtilities::CreateBlueprint → SavePackage
- 返回 `{ name, asset_path, parent_class, created, saved }`

### 2. blueprint_get_event_graph_info（Read）
- 加载 Blueprint → 取 `UbergraphPages[0]`
- 遍历节点找 `UK2Node_Event`，提取 member_name / node_guid / pins
- 返回 `{ exists, graph_name, event_node_count, event_nodes[], has_event_graph }`

### 3. blueprint_add_event_node（Write）
- 加载 Blueprint → GetEventGraph → FindEventNodeByMemberName
- 已存在：复用 node_guid；不存在：CreateEventNode（NewObject + SetExternalMember + AllocateDefaultPins）
- 返回 `{ node_guid, created, event_name, graph_name }` + 完整 pins 信息

### 4. blueprint_add_call_function_node（Write）
- 加载 Blueprint → 定位目标 Graph（graph_name 或默认 EventGraph）
- 校验 function_name：FindFirstObject<UFunction> → FUNC_BlueprintCallable flag
- 创建节点：NewObject<UK2Node_CallFunction> → SetFromFunction → AllocateDefaultPins → AddNode
- 返回 `{ node_guid, function_name, graph_name }` + 完整 pins 信息

### 5. blueprint_connect_pins（Write）
- 加载 Blueprint → 定位目标 Graph → FindNodeByGuid(source/target)
- FindPin(source EGPD_Output) + FindPin(target EGPD_Input)
- Schema->TryCreateConnection(source_pin, target_pin)
- 返回 `{ connected: true }`

### 6. blueprint_compile_save（Write）
- 加载 Blueprint → MarkBlueprintAsModified → CompileBlueprint
- 编译后检查 `BP->Status != BS_Error`
- 可选保存（save=true）：GetOutermost → MarkPackageDirty → SavePackage
- 返回 `{ compiled, saved }`

## 新增错误码（7 个）

| 错误码 | 触发场景 |
|--------|----------|
| GRAPH_NOT_FOUND | Blueprint 无 EventGraph 或指定 Graph 不存在 |
| FUNCTION_NOT_FOUND | function_name 在引擎中未找到 |
| FUNCTION_NOT_CALLABLE | 函数存在但非 BlueprintCallable |
| NODE_NOT_FOUND | node_guid 对应的节点不存在 |
| PIN_NOT_FOUND | pin_name 对应的引脚不存在或方向不匹配 |
| SCHEMA_NOT_FOUND | Graph Schema 为空（不应发生） |
| COMPILE_FAILED | Blueprint 编译后 Status == BS_Error |
