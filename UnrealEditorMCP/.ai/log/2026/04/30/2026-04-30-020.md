### 评估矩阵（8 维度）

| 维度 | 当前状态 | 可选升级路径 | 触发条件 |
|------|----------|--------------|----------|
| 传输协议 | TCP JSON Lines (localhost) | HTTP/SSE、WebSocket | 标准工具接入、多客户端复用 |
| 部署形态 | 外部 Python MCP Server | 插件内 Native Server | 去掉 Python 依赖 |
| 客户端数量 | 单客户端 | 多客户端（会话隔离） | 多个 AI Agent 同时操作 |
| 访问范围 | 127.0.0.1 | LAN/VPN/authenticated remote | 远程团队、CI Runner |
| 安全层 | 无加密 + 可选 token | TLS + 强制认证 | 远程或生产环境 |
| 可观测性 | UE 日志 + Python 日志 | 健康端点 + metrics | 外部监控或运维 |
| 连接模型 | 单 socket + 断连清理 | 连接池 + 心跳 | 长连接可靠性要求高 |
| CI/Headless | 不支持 | -unattended + 无头模式 | CI/CD 自动化 UE |

### 首轮默认推荐

- 保持当前 `localhost TCP JSON Lines + 外部 Python MCP Server`
- 不主动进入 HTTP/SSE 或 Native Server 改造
- 多客户端/远程访问/CI 集成列为"条件触发"
- 可选小修：重连/超时/token 尾项收敛

### 明确不做

- 不预设抛弃 TCP 或重写为 HTTP
- 不在无真实需求时实现会话池
- 不为远程访问引入 PKI/证书管理
- 不做 headless 完整 Editor 自动化框架
