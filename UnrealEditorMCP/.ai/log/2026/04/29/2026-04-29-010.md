## 安全模型设计

```
Dispatcher.Dispatch(action, payload)
  ↓
  Handler = Handlers.Find(action)
  ↓
  Token 非空 && Handler.category ∈ {Write, Dangerous}？
    ├─ 是 → payload._token == Token？
    │       ├─ 匹配 → Execute + AUDIT log
    │       └─ 不匹配 → TOKEN_REQUIRED
    └─ 否 → Execute
```

## 审计日志格式

```
LogMCPDispatcher: AUDIT: action='spawn_actor' category=1 ok=1 error=none
LogMCPDispatcher: AUDIT: action='actor_delete'  category=1 ok=0 error=ACTOR_NOT_FOUND
```

## 使用方式

- **默认模式**：Token 为空字符串，所有操作不受限制（向后兼容）
- **启用保护**：`Server->SetToken("secret")` 后 Write/Dangerous 需携带 `_token`
- **Python 端**：`UEBridgeClient(token="secret")` 自动注入
